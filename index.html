Berita Fakultas

Peresmian Laboratorium Medis Baru FK UI

Fakultas Kedokteran UI meresmikan laboratorium medis baru dengan peralatan mutakhir untuk mendukung pembelajaran mahasiswa kedokteran.

Seminar Kesehatan Global di FK UI

Seminar bertema "Inovasi dalam Kesehatan Global" menghadirkan pakar medis dari berbagai belahan dunia untuk membahas tantangan dan solusi dalam dunia kesehatan.

Kegiatan Mahasiswa

Mahasiswa FK UI Raih Penghargaan di Kompetisi Internasional

Tim mahasiswa FK UI memenangkan penghargaan dalam kompetisi riset medis internasional yang diadakan di Jepang.

Program Bakti Sosial Mahasiswa Kedokteran

Mahasiswa FK UI mengadakan bakti sosial di daerah terpencil untuk memberikan pelayanan kesehatan gratis bagi masyarakat.

Informasi Akademik

Jadwal Pendaftaran Semester Baru

Pendaftaran untuk semester baru dibuka hingga tanggal 15 Februari 2025. Pastikan semua dokumen telah diunggah melalui portal akademik.

Beasiswa bagi Mahasiswa Berprestasi

FK UI membuka pendaftaran beasiswa untuk mahasiswa berprestasi dalam bidang kedokteran. Informasi lebih lanjut dapat dilihat di laman resmi fakultas.
